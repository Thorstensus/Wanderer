# Wanderer

This simple game was created during my study at Green Fox Academy in order to better learn OOP principles before the exam. 

I added my own features, my favorite being an easter egg - try to type "nyanyanyan!" :)

Controls:

Arrows = move

Spacebar = attack

Enjoy!
